## LeafPad
The LeafPad is a 11 key macropad with a rotary encoder, an OLED Display and uses KMK firmware
---------------------------------------------------------------------------------------------
## Features:
- 11 keys for macros
- One rotary encoder for volume controll
- An oled screen for bongo cat (Cuz y not)

## Firmware:
This hackpad uses [KMK](github.com/KMKfw/kmk_firmware/) for everything
This includes:
 - 11 macro keys
 - A rotary encoder for volume control
 - And an oled screen
## BOM:
- 11x Cherry Mx Switches
- 11x DSA Keycaps
- 12x 1N4148 DO-35 Diodes
- 1x XIAO RP2040
- 1x Case
- 1x EC11 Rotary Encoder
- 1x 0.91" 128x32 OLED Display
- 4x M3x16mm screws
- 4x M3x5mx4mm heatset inserts
